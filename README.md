# C++ Pointers Practice

## 📌 Overview

This repository contains my C++ pointer practice programs. The goal of this repository is to build a strong understanding of pointers by implementing and experimenting with different pointer concepts through hands-on coding.

Rather than solving isolated problems, I practiced pointers topic by topic to develop a solid conceptual foundation.

---

## 📚 Topics Covered

### Pointer Fundamentals

* Pointer declaration and initialization
* Pointer reassignment
* Dereferencing pointers
* Address operators (`&` and `*`)
* Void pointers

### Pointers and Arrays

* Relationship between pointers and arrays
* Pointer arithmetic
* Array traversal using pointers
* Passing arrays to functions
* Array limitations

### Const Qualifiers

* Pointer to constant
* Constant pointer
* Constant pointer to constant data

### Pointers and Functions

* Passing pointers to functions
* Returning pointers from functions
* Passing arrays using pointers
* Passing strings to functions

### C-Strings

* String traversal using pointers
* Copying strings using pointers
* Comparing strings
* Dynamic C-string allocation

### Array of Pointers

* Array of pointers
* Array of pointers to strings

### Dynamic Memory Allocation

* `new` operator
* `delete` operator
* Dynamic arrays
* Dynamic C-strings
* `nullptr`
* Memory leaks and how to avoid them

### Copying Objects

* Shallow Copy
* Deep Copy
